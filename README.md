# Agent Skills

[Agent Skills](https://agentskills.io) 是一种简单、开放的格式，用于为智能体赋予新的能力和专业知识。

Skills 是包含指令、脚本和资源的文件夹，智能体可以发现并使用它们来更好地完成特定任务。一次编写，随处使用。

## 快速开始

- [文档](https://agentskills.io) - 指南和教程
- [规范](https://agentskills.io/specification) - 格式详情
- [示例 Skills](https://github.com/anthropics/skills) - 看看能做些什么

本仓库包含规范、文档和参考 SDK。更多示例 skills 请查看[这里](https://github.com/anthropics/skills)。

## 关于

Agent Skills 是由 [Anthropic](https://anthropic.com) 维护的开放格式，欢迎社区贡献。
